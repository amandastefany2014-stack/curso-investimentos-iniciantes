# curso-investimentos-iniciantes
O curso foca na organização pessoal e planejamento financeiro. Ele é estruturado para fornecer tanto a base técnica quanto o preparo psicológico necessários para a autonomia financeira
O tema escolhido para este caderno temático foi Investimentos básicos para iniciantes. O interesse por esse assunto surgiu pela importância da educação financeira na vida das pessoas e pela necessidade de aprender a investir de forma consciente e segura.

Quais são os melhores investimentos para quem está começando?
Quais são os principais conceitos que um iniciante precisa aprender antes de investir?
